# Masakhe Partners — Design System

## Brand Identity
Premium wealth-building opportunity for South African entrepreneurs. Masakhe's trusted SMME platform as the product, powerful MLM/reseller network as the vehicle.

## Typography
- **Display**: Cormorant Garamond (bold headlines, section titles)
- **Body**: DM Sans (clean, readable, modern)
- **Accent/Numbers**: Space Mono (stats, earnings, tier codes)
- Import from Google Fonts

## Color Palette
- **Gold Primary**: #D4AF37
- **Gold Light**: #F5C518
- **Gold Dim**: #A8891B
- **Black**: #0A0A0A
- **Dark 1**: #111111
- **Dark 2**: #1A1A1A
- **Dark 3**: #222222
- **Dark Surface**: #2A2A2A
- **Border Gold**: rgba(212,175,55,0.25)
- **Text Primary**: #FFFFFF
- **Text Secondary**: rgba(255,255,255,0.7)
- **Text Muted**: rgba(255,255,255,0.45)

## CSS Variables (override in styles.css)
Dark theme by default (premium feel). Gold = primary. Black/dark = background.

## Layout
- Full-width dark sections with alternating depths
- Gold gradients, geometric angular dividers
- Centered editorial headings with generous spacing
- No rounded cards — use sharp or slight radius for premium feel
- Asymmetric hero layout

## Motion
- Staggered entrance animations on scroll (IntersectionObserver)
- Gold shimmer on key CTAs
- Count-up animations on stats

## Components Style
- Buttons: gold filled (primary), gold border (secondary), sharp corners
- Cards: dark surface, gold border-left accent
- Badges: gold outline pill
- Dividers: gold gradient lines
